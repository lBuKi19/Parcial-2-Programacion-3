# Parcial-2-Programacion-3

Este proyecto consiste en una aplicación de consola desarrollada en Java utilizando JPA (Jakarta Persistence API) para la gestión de datos. Permite realizar operaciones CRUD (alta, baja lógica, modificación y consulta) sobre entidades como Categoría y Producto, utilizando repositorios propios para encapsular la lógica de acceso a datos.

El menú por consola permite navegar las distintas operacion CRUD sobre categorias y productos, con mensajes de alerta cuando se efectúan los cambios correctamente, asi como cuando hay errores.
